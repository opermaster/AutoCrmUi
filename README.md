# Frontend for the [AutoCrm Backend](https://github.com/opermaster/AutoCrmApi)

## Overview

This project provides a web interface for managing an auto repair shop. It allows administrators, managers, and technicians to efficiently manage workshop operations, including employees, services, parts, and repair orders.

---

## Admin Panel

The Admin panel allows full system management, including:

* Managing users and assigning roles based on their position
* Managing services provided by the workshop
* Managing parts used in services
* Managing repair orders

## Technician (Master) Panel

The Technician panel is intended for service specialists responsible for completing repair work. It allows technicians to:

* View assigned orders
* Update order statuses
* Mark services as completed
* Add or edit service comments
